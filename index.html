First of all, you need to setup i18n at least with { baseDir: 'dir' } parameter
the defaults is: { language: 'en', defaultLanguage: 'en', auto: false }

There are 2 kind of files:
1) resource.LANGUAGE.i18n
2) preferences.i18n (this is global preferences, like the name suggest, this translation file has preference over normal resource file)

You can use i18n directly on html elements setting i18n as class, and i18n-key="KEY_NAME" attribute, then on document load i18n.applyOnDocument();
You can also use i18n.get('KEY_NAME') on javascript, returning a string or null value
